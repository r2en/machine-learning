# Grid Search: グリッドサーチ

ハイパーパラメータの探索空間を格子状 (グリッド) に区切り<br>
交点となるハイパーパラメータの組み合わせについて、すべて調べるという方法です。<br>

正答率の高いものほど大きい丸で表しています。<br>
このように、すべての格子点 (ハイパーパラメータの組み合わせ) でモデルを作り、<br>
最も良い評価値となるハイパーパラメータを選択するというのがグリッドサーチです。<br>
